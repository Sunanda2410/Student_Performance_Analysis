# Student Performance Analysis

This project is part of my data analysis internship. In this project, I worked with a student performance dataset to understand how different factors can affect students' final grades.

## What I Did

* Loaded the dataset using Pandas
* Explored the data and checked for missing values
* Cleaned and prepared the dataset for analysis
* Analyzed different factors related to student performance
* Created visualizations using Matplotlib and Seaborn
* Documented the complete process in a Jupyter Notebook

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Files in This Repository

* student_analysis.ipynb – Complete analysis and visualizations
* student-mat.csv – Dataset used for the project
* README.md – Project description

## Key Learnings

Through this project, I learned how to clean data, perform exploratory data analysis (EDA), create visualizations, and draw meaningful insights from real-world datasets.

## Conclusion

This project helped me strengthen my Python and data analysis skills by working on a practical dataset and presenting the findings through charts and visualizations.
